## Overview Aplikasi

**GlowUp Clinic App** adalah aplikasi manajemen klinik kecantikan yang komprehensif, dibangun dengan Flutter. Aplikasi ini mendukung tampilan responsif untuk **phone** dan **tablet**, dengan fokus pada kemudahan operasional sehari-hari klinik.

### Tech Stack
- **Framework**: Flutter 3.x
- **State Management**: BLoC (Business Logic Component)
- **Backend**: Laravel API (REST)
- **Database**: MySQL (via Laravel)
- **Authentication**: Laravel Sanctum (Token-based)

### Platform Support
- Android
- iOS
- Tablet (Android/iPad)

### 1. Authentication (Login/Logout)

**Lokasi**: `lib/presentation/auth/`

**Komponen**:
- `LoginBloc` - Handle login logic
- `LogoutBloc` - Handle logout logic
- `SplashPage` - Check authentication status
- `LoginPage` - Login form

**API Endpoints**:
```
POST /api/v1/login     - Login dengan email & password
POST /api/v1/logout    - Logout & hapus token
GET  /api/v1/profile   - Ambil data user yang login
```

**Fitur**:
- Form validation (email, password)
- Remember me (token disimpan di SharedPreferences)
- Auto-login jika token masih valid
- Session expiry handling (401)
